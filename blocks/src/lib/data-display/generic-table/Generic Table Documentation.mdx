# Generic Table Component

## Overview

The `GenericTable` is an infinitely flexible React component designed to render any structured data with advanced features like sorting, filtering, pagination, and column visibility.

## Features

- Completely type-agnostic data rendering
- Fully customizable through generic type parameters
- Advanced table interactions:
  - Sorting
  - Filtering
  - Pagination
  - Column visibility toggle
- Responsive design
- TypeScript generic support

## Installation

```bash
npm install @cash-compass/blocks @tanstack/react-table
```

## Usage Examples

### Basic Usage with Different Data Types

```tsx
import { GenericTable } from '@cash-compass/blocks';
import { ColumnDef } from '@tanstack/react-table';

// User Data Example
type User = {
  id: number;
  name: string;
  email: string;
  role: string;
};

const userColumns: ColumnDef<User>[] = [
  {
    accessorKey: 'id',
    header: 'ID',
  },
  {
    accessorKey: 'name',
    header: 'Name',
  },
  {
    accessorKey: 'email',
    header: 'Email',
  },
  {
    accessorKey: 'role',
    header: 'Role',
  },
];

const users: User[] = [
  { id: 1, name: 'John Doe', email: 'john@example.com', role: 'Admin' },
  { id: 2, name: 'Jane Smith', email: 'jane@example.com', role: 'User' },
];

// Product Inventory Example
type Product = {
  sku: string;
  name: string;
  price: number;
  stock: number;
};

const productColumns: ColumnDef<Product>[] = [
  {
    accessorKey: 'sku',
    header: 'SKU',
  },
  {
    accessorKey: 'name',
    header: 'Product Name',
  },
  {
    accessorKey: 'price',
    header: 'Price',
    cell: (info) => `$${info.getValue<number>().toFixed(2)}`,
  },
  {
    accessorKey: 'stock',
    header: 'Stock',
  },
];

const products: Product[] = [
  { sku: 'TECH-001', name: 'Wireless Headphones', price: 129.99, stock: 50 },
  { sku: 'TECH-002', name: 'Smart Watch', price: 199.99, stock: 25 },
];

function DataDisplayComponent() {
  return (
    <>
      <GenericTable
        data={users}
        columns={userColumns}
        title="User Management"
        enableSorting
        enableFiltering
        filterColumn="name"
      />

      <GenericTable
        data={products}
        columns={productColumns}
        title="Product Inventory"
        enableSorting
        enablePagination
      />
    </>
  );
}
```

## Column Definition Flexibility

Use `@tanstack/react-table`'s `ColumnDef` to define columns with:

- `accessorKey`: Data property to display
- `header`: Column header text
- `cell`: Custom cell rendering function
- `meta`: Additional metadata for styling or behavior

## Accessibility

- Keyboard navigable
- Screen reader friendly
- Semantic HTML table structure
- WCAG compliant interactions

## Performance Considerations

- Optimized for various dataset sizes
- Minimal re-renders
- Efficient data manipulation with `@tanstack/react-table`

## Customization

Highly customizable through:

- Generic type support
- Flexible column definitions
- Styling props
- Event handlers
- Feature toggles

## Dependencies

- React
- @tanstack/react-table
- Tailwind CSS
- Lucide React (for icons)

## Best Practices

- Use TypeScript for type safety
- Define clear, concise column definitions
- Implement custom cell renderers for complex data
- Consider performance with large datasets
- Leverage type generics for maximum flexibility
